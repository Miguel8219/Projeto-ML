# Projeto-ML
Repositório com exercícios práticos e projetos desenvolvidos ao longo da disciplina de Machine Learning. Este material abrange etapas fundamentais do fluxo de trabalho em aprendizado de máquina, incluindo pré-processamento de dados, análise exploratória, engenharia de atributos, treinamento de modelos, avaliação de desempenho e comparação de resultados.

Os temas explorados incluem aprendizado supervisionado, regressão, classificação, validação de modelos, ajuste de hiperparâmetros e utilização de bibliotecas Python como Pandas, NumPy, Scikit-learn, Matplotlib e Seaborn.

O principal objetivo deste repositório é documentar minha trajetória de aprendizado, consolidar conhecimentos práticos e demonstrar experiência aplicada em Machine Learning e Ciência de Dados. A descrição de cada exercício está detalhada em cada notebook.
